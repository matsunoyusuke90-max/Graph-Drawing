# ƒ(x) plotter — 関数グラフ描画アプリ

数式を入力すると y = f(x) のグラフを描くブラウザアプリです。複数の関数を重ねて表示でき、ドラッグとスクロールで自由に拡大・移動できます。
ダウンロードして個人用にお使いください。

## 使い方

`index.html` をブラウザで開くだけ。

- サイドバーに数式（例：`sin(x)`、`x^2 / 4`）を入力すると即座に描画されます。
- 関数は複数追加でき、色分け表示・表示/非表示の切り替え・削除ができます。
- **グラフ操作** — ドラッグで移動（pan）、スクロールで拡大縮小（zoom）。`＋`／`−`ボタンや `⟲`（リセット）も利用できます。
- カーソル位置の座標・関数値を読み取れます。

## 数式の記法

`math.js` の記法が使えます（`sin`, `cos`, `sqrt`, `^`（べき乗）, `/`, `*` など）。入力欄は不正な式を赤くハイライトし、エラー内容を表示します。

## 技術

- 単一HTML。**math.js** をCDNから読み込み、式の解析・評価を行います。
- グラフ描画は HTML5 Canvas。
- CDNを利用するため、初回利用時はネット接続が必要です。
